Este programa consiste em:
* Você escolhe a quantidade de números aleatórios que seram gerados.
* O programa gera um número aleatório dentro do intervalo escolhido pelo usuário.
* Para cada um dos 3 jogadores setados manualmente é gerado também um número aleatório.
* Após os dois ultimos passos é feita uma comparação entre o número a ser encontrado e o número dos jogadores.
* O programa vai gerar números aleatórios a cada rodada até que seja encontrado o número correrpondente.
* Assim que encontrado o número correto o programa exibe os ganhadores e encerra.
